# Dotfiles, keymaps and misc for flyingpeakock

My dotfiles for tmux, vim and zsh. Also a script to symlink the
correct files. There is also a script to download vim, tmux and zsh
if they aren't already present.
There is also a script to change the keyboard to Swedish Colemak    

    git clone --single-branch --branch pi --recurse-submodules https://github.com/flyingpeakock/dotfiles.git ~/.dotfiles
