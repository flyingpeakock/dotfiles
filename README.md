# Dotfiles, keymaps and misc for flyingpeakock

My dotfiles for tmux, vim and zsh. Also a script to symlink the
correct files. There is also a script that installs my commonly used
programs, inluding i3 and my i3 dotfiles.
There is also a script to change the keyboard to Swedish Colemak    

    git clone --recurse-submodules https://github.com/flyingpeakock/dotfiles.git ~/.dotfiles
